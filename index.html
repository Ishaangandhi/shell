Developer tools matter

So why are you using a terminal designed in the 1970's?
Ishan is packed with features to unleash your productivity.

Save and Organize Commands

Ishan was built to optimize common workflows on the terminal. Keep your commonly used commands in a single location, and run them together. Effortlessly. Don't worry about what that third flag was. Save your notebooks so you can keep coming back to them.

SSH Integration

Easily configure your notebook to connect to a remote machine. Once a machine is added, you can connect to it again from any notebook with a single click.

Native File Navigation

No more "cd ../..; ls; cd ..; ls;". Navigate around your Mac the way it was supposed to be.

Power-user friendly

Ergonomic keybindings for frustration-free programming.

⌘+T New tab
⌘+O Open
⌘+S Save
⌘+Shift+S Save as..
⌘+W Close tab
⌃⌥↑ Move cell up
⌃⌥↓ Move cell down
⇧+enter Run cell
⌘+D Delete cell
⌘+N New cell
⌘+R Connect remote machine
⌘+Shift+R Disconnect remote machine
